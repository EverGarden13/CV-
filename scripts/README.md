# VisionMate-Lite Scripts

This folder contains utility and validation scripts.

## Available Scripts

### `validate_system.py`
System validation script that:
- Checks all dependencies
- Validates camera functionality
- Tests core system components
- Provides diagnostic information

### `run_evaluation_example.py`
Evaluation runner that:
- Executes performance tests
- Generates evaluation reports
- Benchmarks system components

## Usage

Run scripts from the project root:
```bash
python scripts/validate_system.py
python scripts/run_evaluation_example.py
```
