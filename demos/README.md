# VisionMate-Lite Demos

This folder contains demonstration scripts and examples.

## Available Demos

### `demo_system.py`
Basic system demonstration showing core functionality including:
- Object detection
- OCR text reading
- Audio feedback

### `demo_scene_integration.py`
Advanced demonstration featuring:
- Scene classification integration
- Enhanced object detection with scene context
- Multi-modal feedback system

## Running Demos

Make sure you have installed all requirements:
```bash
pip install -r ../requirements.txt
```

Run demos from the project root:
```bash
python demos/demo_system.py
python demos/demo_scene_integration.py
```
